## 4. Installation steps

### From VS Code

1. Go to extension part on VS Code activy bar;
2. Search for Lucid Volcano theme;
3. Install and **Enjoy!**

### Via CLI:

```
code --install-extension marcosgabriel.lucid-volcano
```
